# Pig-Game FPS

A browser-based first-person 3D parkour shooter built with Three.js.

## Gameplay

- Climb a central vertical tower using parkour routes.
- Fight flying winged zombie pigs as you ascend.
- Use a bazooka that fires explosive missiles.
- Survive with jump, double jump, and fall-damage risk.
- Defeat the Pig Lord boss on the summit platform.

## Controls

- Move: `W A S D`
- Sprint: `Shift`
- Jump: `Space`
- Double jump: `Space` in midair
- Aim: mouse
- Fire bazooka missile: left click

## Run locally

Serve the folder over HTTP and open in a browser.

Example:

```bash
cd /workspaces/game-sample
python3 -m http.server 8000
```

Then open `http://localhost:8000`.
